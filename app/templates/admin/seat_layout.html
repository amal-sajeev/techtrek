How to build your layout

  1. Set the grid size with + Row / + Column (or − Row / − Column to shrink). Every auditorium can have a different shape.
  2. Choose a seat type (Standard, VIP, Accessible, Reserved, or Aisle). Click or drag to paint; right-click or Eraser to clear.
  3. Use Fill all with [type] to fill the grid, or click a row letter or column number to fill that row/column. Put aisles wherever you want.
  4. Right-click a row letter or column number for options to insert before/after, delete, or clear that row/column from anywhere in the grid.
  5. Shift+click a row letter or column number to add a walkway gap (extra space between sections). You can also click the thin handles between rows. Gaps appear on the public booking page.
  6. Save layout when done. Grid size, stage width, and gaps are all saved.
